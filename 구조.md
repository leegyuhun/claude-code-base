          ├── next.md                 ← /next: 다음 실행할 에이전트 명령어 안내
          ├── rollback.md             ← /rollback: 특정 PHASE로 되돌리기
          ├── sprint-log.md           ← /sprint-log: 현재 스프린트 종합 요약
          └── debt.md                 ← /debt: Tech Debt 종합 보고

  commands vs skills 차이:
  - commands/ → 긴 작업 자동화 (에이전트 수준 실행, 코드 수정 가능)
  - skills/ → 상태 조회/안내만 (읽기 전용, 출력 후 종료)
