## 목적 & 배경

1. 프린터 IP가 변경됨으로써 모듈에서 프린터에 접속하지 못하는 케이스가 발생.
2. TCP 포트스캔(9100)으로 IP를 찾으려 했으나, L3 스위치에서 포트스캔을 차단하는 케이스 발생.
3. 서브넷이 다른 네트워크에 연결된 프린터도 탐색할 수 있어야 함.
4. 탐색 방식: **SNMP (UDP 161)**를 사용하여 프린터를 발견 (L3 스위치에서 차단될 가능성이 낮음).

---

## 핵심 기능 목록

### 1. SNMP 기반 프린터 탐색
- UDP 161 포트로 SNMP GET/WALK 요청을 보내 프린터 여부를 판별
- SNMP v1/v2c 지원
- Community String은 UI에서 설정 가능 (기본값: `public`)
- 탐색 성공 조건: SNMP OID `1.3.6.1.2.1.25.3.2.1.2` (hrDeviceType = printer) 응답 확인

### 2. 스캔 대상 네트워크 범위 지정
- **자동 감지**: 로컬 PC에 연결된 네트워크 인터페이스 정보를 읽어 스캔 대상 서브넷을 자동으로 도출 (예: 192.168.1.0/24)
- **수동 입력**: 추가 서브넷을 사용자가 직접 입력 (CIDR 표기 또는 시작IP~끝IP 범위)
- 여러 서브넷 동시 스캔 가능

### 3. 프린터 목록 표시
- 탐색된 프린터 정보를 ListView에 표시
  - IP 주소
  - 프린터 이름 (SNMP sysName OID: `1.3.6.1.2.1.1.5.0`)
  - 모델명 (SNMP hrDeviceDescr OID: `1.3.6.1.2.1.25.3.2.1.3`)
  - 응답 시간 (ms)
  - 상태 (온라인/오프라인)
- 항목 선택 후 "접속" 버튼으로 해당 IP로 프린터 연결 전환 가능

### 4. 실시간 로그
- 우측 패널에 스캔 진행 상황을 실시간으로 출력
  - 스캔 시작/완료, 각 IP 응답 여부, 에러 메시지 등
- 로그 저장 버튼 (텍스트 파일 export)

### 5. DLL Export 인터페이스
기존 모듈에서 호출할 수 있도록 stdcall 함수를 DLL로 export:

```pascal
// 프린터 스캔 창 표시 (모달)
// 반환값: 선택된 프린터 IP (없으면 빈 문자열)
function ShowPrinterScanner(
  hOwner: HWND;                   // 부모 윈도우 핸들
  pszCommunity: PAnsiChar;        // SNMP Community String (nil이면 'public' 사용)
  pszSubnets: PAnsiChar;          // 추가 서브넷 목록 (세미콜론 구분, nil 가능)
  pszSelectedIP: PAnsiChar;       // [out] 선택된 IP 버퍼
  nBufLen: Integer                // pszSelectedIP 버퍼 크기
): Integer; stdcall;              // 0=취소, 1=선택됨

// 백그라운드 스캔 실행 (비동기)
// 결과는 콜백으로 전달
procedure StartPrinterScan(
  pszCommunity: PAnsiChar;
  pszSubnets: PAnsiChar;
  pfnCallback: TPrinterFoundCallback
); stdcall;
```

## 기술 스택

- **언어/IDE**: Delphi 2007 (Win32)
- **SNMP**: Indy 10 (IdSNMP) 또는 직접 UDP 소켓 구현
- **멀티스레딩**: TThread (스캔은 백그라운드 스레드에서 실행, VCL 접근은 Synchronize 사용)
- **네트워크 인터페이스 감지**: WinAPI `GetAdaptersInfo` (IpHlpApi)
- **출력 형태**: EXE (독립 실행) + DLL (stdcall export)

---

## MVP 완료 기준

1. **10초 이내** 스캔 완료 (단일 /24 서브넷 기준, 타임아웃 1초, 병렬 스캔)
2. SNMP 응답이 있는 프린터가 목록에 정상 표시됨
3. 목록에서 프린터 선택 후 "접속" 클릭 시 해당 IP 반환
4. DLL export 함수 `ShowPrinterScanner` 호출 시 모달 창 표시 및 선택 IP 반환
5. 로컬 네트워크 서브넷 자동 감지 동작
6. 수동 서브넷 입력 후 스캔 동작

## 제약 조건

- Delphi 2007 (Unicode 미지원, AnsiString 기반)
- Win32 전용 (64비트 불필요)
- SNMP v1/v2c만 지원 (v3 불필요)
- 외부 설치 불필요 (인스톨러 없이 EXE/DLL 단독 배포)
- 서드파티 라이센스가 필요한 컴포넌트 사용 금지 (Indy는 무료)
